![Resolução](https://user-images.githubusercontent.com/51630262/196067751-775870bb-156f-40ab-83e7-34156fcd29a4.png)

1. Para a criação de um diretório é preciso utilizar o comando `mkdir <nome do diretório>`, para que seja feita a navegação é necessário utilizar o `cd <caminho para o diretório>`. 

![1](https://user-images.githubusercontent.com/51630262/199841344-07dfcafc-facd-45a5-ac37-5b0e0b1f0b97.png)

2. Uma forma de criar um arquivo e adicionar um conteúdo tudo pelo terminal, é utilizar o `cat > <arquivo.tipo>`, após fazer esse comando é só colocar as palavras desejadas, lembrando que ao utilizar `>` se já existir um arquivo e ele tiver conteúdo, o conteúdo será substituido, se quiser adicionar um conteúdo e manter aqueles que já tem no arquivo é possível utilizar o `>>`, ao invés do `>`.

![2](https://user-images.githubusercontent.com/51630262/199841354-c3f2d721-6711-425f-97ae-7615ff600992.png)

3. Como citado no item dois, para adicionar conteúdo em um arquivo que já tem conteúdo dentro, sem apagar, só precisa utilizar o `>>`, da seguinte forma `cat >> nome do arquivo`, após o comando é preciso só adicionar as novas palavras.

![3](https://user-images.githubusercontent.com/51630262/199841359-1b6fcdc4-3730-482d-8dab-e44c285b6fde.png)

4. Para saber a quantidade de linhas, é preciso utilizar o comando `wc -l <arquivo>`, o `wc` é um comando que informa a quantidade de linhas, palavras e caracteres, ao colocar o `-l`, indica que a única informação que estou pedindo é o número de linhas que existe naquele arquivo
