1. Crie uma constante chamada base e atribua a ela o valor 5.
2. Crie uma constante uma chamada heigth e atribua a ela o valor 8.
3. Crie uma constante chamada area e atribua a ela o resultado da multiplicação da base pela heigth. Dica: lembre-se de usar sempre o console.log() para imprimir as variáveis e checar os resultados das operações!
4. Crie uma constante chamada perimeter e atribua a ela a soma de todos os lados do retângulo.

![Resolução](https://user-images.githubusercontent.com/51630262/196067751-775870bb-156f-40ab-83e7-34156fcd29a4.png) 

1. Para criação dessa constante é preciso utilizar o `const` e o nome que irá ter essa constante (base), depois é preciso atribuir o valor 5 . Lembrando que constantes, não podem ser alteradas depois de declaradas.

```
const base = 5;
```

2. Para criação dessa constante é preciso utilizar o `const` e o nome que irá ter essa constante (heigth), depois é preciso atribuir o valor 8. 

```
 const heigth = 8;
```

3. Para criação dessa constante é preciso utilizar o `const` e o nome que irá ter essa constante (area), depois é preciso atribuir para essa constante a formula que calcula a area de um triangulo.

```
const area = base * heigth;
```

4. Para criação dessa constante é preciso utilizar o `const` e o nome que irá ter essa constante (perimeter), depois é preciso atribuir para essa constante a formula que calcula o perimetro, após realizar as declarações, para conferir se está tudo correto, pode se utilizar o `console.log();`, para imprimir o valor das constantes.

```
const perimeter = 2 * (base + heigth);

console.log(base);
console.log(heigth);
console.log(area);
console.log(perimeter);
```
